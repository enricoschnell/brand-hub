# CASAGO Brand Hub

Internal brand guidelines and asset management for [CASAGO GmbH](https://casago.de).

**Live**: [brand.casago.de](https://brand.casago.de)

## Pages

- **Home** — Animated Wortmarke with cinematic SVG path-drawing
- **Logo** — All variants (Black, White, Outline) in SVG, PNG, PDF, EPS
- **Farben** — Complete color palette from the Figma design system
- **Typografie** — Aeonik Pro specimen, type scale, desktop font downloads
- **Email Signatur** — On-brand signatures with live preview, Light/Dark toggle, HTML export

## Stack

Next.js 14 · shadcn/ui · Tailwind CSS · Vercel

## Development

```bash
npm install
npm run dev
```

## Design

- **Hub UI**: Fully dark (Lovable/Linear-style), TikTok Sans Variable
- **Brand Font**: Aeonik Pro — Regular (400) + Medium (500) only
- **Max Radius**: 12px — no rounded pill shapes
- **Colors**: Borderless square swatches (eBay/Wise pattern)
- **Signature**: WCAG AA contrast-audited, two-tier color system
